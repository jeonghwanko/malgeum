# 프리미엄 구독

맑음 프리미엄의 혜택과 가격을 안내합니다.

---

## 무료와 프리미엄의 차이는 무엇인가요?

| 기능 | 무료 | 프리미엄 |
|------|------|---------|
| 행동 추천 카드 | ✅ 전체 제공 | ✅ 전체 제공 |
| 출퇴근 날씨 비교 | ✅ | ✅ |
| 주간 예보 | ✅ | ✅ |
| 날씨 알림 | ✅ | ✅ |
| AI 맑음이 채팅 | 하루 10회 | **하루 100회** |
| 아트 테마 | 기본+클래식 8가지 | **8가지 추가 (총 16가지)** |
| 위젯 | ✅ | ✅ |

핵심 날씨 기능은 무료로 모두 사용할 수 있습니다.  
프리미엄은 8가지 추가 아트 테마와 AI 채팅 하루 100회를 원하는 분께 적합합니다.

## 가격은 얼마인가요?

| 플랜 | 가격 |
|------|------|
| 월간 구독 | 월 3,300원 |

---

## 프리미엄 구독 방법

1. 맑음 앱 → 설정 탭 → **프리미엄으로 업그레이드**
2. 결제 완료 후 즉시 활성화됩니다

결제는 구글 플레이 또는 앱스토어를 통해 이루어집니다.

---

## 구독 취소 방법

### Android (Google Play)
Google Play → 프로필 → 결제 및 구독 → 구독 → 맑음 → 구독 취소

### iOS (App Store)
설정 → Apple 계정 → 구독 → 맑음 → 구독 취소

취소 후에도 현재 결제 기간이 끝날 때까지 프리미엄 기능을 사용할 수 있습니다.

---

## 환불은 가능한가요?

결제 후 7일 이내에는 환불을 요청할 수 있습니다.  
구글 플레이 또는 앱스토어 고객센터를 통해 환불을 신청하세요.

---

## 여러 기기에서 사용할 수 있나요?

같은 계정(구글 플레이 또는 앱스토어)으로 로그인한 기기에서 모두 사용할 수 있습니다.  
구독 복원: 설정 → 프리미엄 → 구독 복원
