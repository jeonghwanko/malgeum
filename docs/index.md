# 맑음 앱 가이드

**맑음(Malgeum)** 은 날씨 숫자를 행동으로 번역해주는 AI 날씨 앱입니다.  
"오늘 우산 필요해?", "뭐 입고 나가지?", "미세먼지 심해?" — 맑음이 답해드립니다.

## 맑음의 핵심 가치

### 숫자가 아닌 행동으로 답합니다

| 기존 날씨 앱 | 맑음 |
|-------------|------|
| 기온 17도, 강수확률 60% | "가디건 챙기고 우산 필수예요" |
| 미세먼지 81 μg/m³ | "마스크 꼭 쓰세요 (나쁨)" |
| UV 지수 8 | "자외선 강해요 — 선크림 바르세요" |

### 출근과 퇴근 날씨를 동시에 비교합니다

아침에 입고 나간 옷이 저녁에 맞지 않는 경험, 맑음이 미리 알려드립니다.  
나갈 때 vs 올 때 기온·강수·체감온도를 나란히 비교해 옷차림 전략을 제시합니다.

### AI에게 자연어로 물어보세요

"오늘 빨래 널어도 돼?", "주말에 등산 가도 될까?", "이번 주 가장 맑은 날은?"  
맑음이가 현재 날씨 데이터를 바탕으로 감각적인 언어로 답해드립니다.
