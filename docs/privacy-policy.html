개인정보처리방침

맑음 (Malgeum) | 최종 수정일: 2026년 4월 6일 | 시행일: 2026년 4월 6일

1. 개요

슈퍼빌런랩스(이하 "회사")은 맑음(이하 "앱") 이용자의 개인정보를 중요시하며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 개인정보처리방침은 앱이 어떤 정보를 수집하고 어떻게 사용하는지 설명합니다.

2. 수집하는 정보

2.1 사용자가 직접 제공하는 정보

항목설명필수 여부
닉네임앱 내 표시 이름선택
출퇴근 시간출퇴근 날씨 비교 기능선택
건강 프로필알레르기, 운동 선호, 옷차림 스타일선택
  • 이메일, 전화번호, 실명 등 개인식별정보는 수집하지 않습니다.
  • 회원가입 및 로그인 절차가 없습니다.

2.2 위치 정보

항목설명필수 여부
현재 위치 (위도/경도)해당 지역의 날씨 정보 조회필수
  • 위치 정보는 날씨 API 호출 시에만 사용되며, 별도로 서버에 저장하지 않습니다.
  • 사용자가 저장한 위치 목록은 기기 로컬 저장소에만 보관됩니다.

2.3 앱이 기기에 저장하는 정보

아래 정보는 사용자의 기기(로컬 저장소)에만 저장되며, 외부 서버로 전송되지 않습니다.

  • 닉네임, 출퇴근 시간, 건강 프로필
  • 저장한 위치 목록
  • 날씨 캐시 데이터 (10분 TTL)
  • 아트 테마 설정
  • 알림 설정
  • 추천 적중률 피드백 기록 (최근 90일)
  • 온보딩 완료 여부

2.4 제3자 서비스를 통해 자동 수집되는 정보

앱은 아래 제3자 서비스를 사용하며, 해당 서비스가 기술적 목적으로 정보를 수집할 수 있습니다.

OpenWeatherMap (날씨 데이터)

  • 위도/경도 (날씨 조회 목적)
  • OpenWeatherMap 개인정보처리방침

Firebase (앱 설정 관리)

  • 앱 인스턴스 ID
  • 기기 정보 (모델명, OS 버전)
  • 앱 버전 정보
  • Firebase 개인정보처리방침

RevenueCat (인앱 결제)

  • 익명 사용자 ID (기기 기반 자동 생성)
  • 구독 상태 정보
  • RevenueCat 개인정보처리방침

Sentry (오류 추적)

  • 앱 오류 로그, 스택 트레이스
  • 기기 정보 (모델명, OS 버전)
  • 개발 모드에서는 비활성화
  • Sentry 개인정보처리방침

3. 정보의 이용 목적

목적설명
날씨 정보 제공현재/시간별/주간 날씨, 대기질, 자외선 정보 조회
행동 추천날씨 기반 옷차림, 우산, 운동 등 맞춤 추천
출퇴근 비교출근/퇴근 시간대 날씨 비교
알림 발송날씨 변화, 미세먼지, 비 예보 등 로컬 알림
구독 관리RevenueCat을 통한 프리미엄 구독 처리
앱 안정성Sentry를 통한 오류 감지 및 개선
앱 설정 관리Firebase를 통한 원격 설정 업데이트

4. 정보의 보관 및 삭제

  • 모든 사용자 데이터는 기기 로컬 저장소에만 보관됩니다.
  • 앱을 삭제하면 모든 데이터가 즉시 삭제됩니다.
  • 별도의 서버에 사용자 데이터를 저장하지 않으므로, 앱 삭제 외에 추가적인 삭제 요청이 필요하지 않습니다.
  • 날씨 캐시 데이터는 10분 후 자동 만료됩니다.
  • 피드백 기록은 90일 경과 시 자동 삭제됩니다.

5. 정보의 제3자 제공

회사는 사용자의 개인정보를 제3자에게 직접 제공하지 않습니다. 다만, 아래 제3자 서비스가 자체 정책에 따라 기술적 데이터를 수집할 수 있습니다.

제3자목적수집 항목
OpenWeatherMap날씨 데이터 제공위도/경도
Firebase앱 설정 배포앱 인스턴스 ID, 기기 정보
RevenueCat구독 관리익명 사용자 ID, 구독 상태
Sentry오류 추적오류 로그, 기기 정보

6. 아동의 개인정보 보호

앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만 이용자의 사용이 확인될 경우, 해당 데이터를 즉시 삭제하겠습니다.

7. 사용자의 권리

  • 데이터 확인: 앱 내 설정 화면에서 저장된 정보를 확인할 수 있습니다.
  • 데이터 삭제: 앱을 삭제하면 모든 데이터가 기기에서 완전히 삭제됩니다.
  • 위치 권한 철회: 기기 설정에서 위치 권한을 언제든지 해제할 수 있습니다.
  • 알림 해제: 앱 내 설정 또는 기기 설정에서 알림을 비활성화할 수 있습니다.

8. 보안

  • 사용자 데이터는 기기 내부 저장소에만 보관되어 외부 접근이 제한됩니다.
  • 외부 서버와의 모든 통신은 HTTPS를 통해 암호화됩니다.
  • API 키는 환경변수로 관리되며, 소스 코드에 포함되지 않습니다.

9. 개인정보처리방침의 변경

본 방침이 변경될 경우, 앱 업데이트 또는 스토어 페이지를 통해 공지합니다. 변경 사항은 공지일로부터 7일 후 시행됩니다.

Privacy Policy (English Summary)

Malgeum is a weather-based lifestyle app that provides action recommendations (clothing, umbrella, exercise) instead of just numbers. We collect minimal data:

  • Location: Used solely for weather API calls; not stored on any server.
  • Local only: Nickname, commute times, health profile, theme settings, and feedback are stored only on your device.
  • No account required: No email, phone number, or real name is collected.
  • Third-party services: OpenWeatherMap (weather data), Firebase (app configuration), RevenueCat (in-app purchases), and Sentry (error tracking) may collect device information per their own privacy policies.
  • Data deletion: Uninstalling the app deletes all local data immediately.
