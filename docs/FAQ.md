# 자주 묻는 질문 (FAQ)

---

## 날씨 기능

### 강수확률 몇 %부터 우산 알림이 오나요?

강수확률 30% 이상이면 "우산 권장" 카드가 표시됩니다.  
50% 이상이면 "우산 필수"로 등급이 올라가고, 알림이 설정된 경우 푸시 알림도 전송됩니다.

### 미세먼지 기준이 어떻게 되나요?

| 등급 | PM2.5 (μg/m³) | 행동 |
|------|--------------|------|
| 좋음 | 0–15 | 마스크 불필요 |
| 보통 | 16–35 | 민감군 주의 |
| 나쁨 | 36–75 | 마스크 권장 |
| 매우 나쁨 | 76 이상 | 마스크 필수, 외출 자제 |

### 날씨 데이터는 얼마나 자주 갱신되나요?

앱을 열 때마다 자동 갱신됩니다.  
백그라운드에서는 위젯이 30분마다 갱신됩니다.

### 날씨 데이터는 어디서 가져오나요?

OpenWeatherMap API를 사용합니다.  
현재 날씨, 시간별 예보, 5일 예보, 대기질(미세먼지·초미세먼지·자외선·꽃가루)을 제공합니다.

### 몇 일 후 날씨까지 볼 수 있나요?

최대 5일 예보를 제공합니다. 6일 이상의 날씨는 정확도가 낮아 제공하지 않습니다.

### 오프라인에서도 사용할 수 있나요?

마지막으로 불러온 날씨 데이터를 오프라인에서 볼 수 있습니다.  
실시간 업데이트는 인터넷 연결이 필요합니다.

## AI 맑음이

### AI 맑음이에게 어떤 질문을 할 수 있나요?

날씨와 관련된 모든 질문을 자연어로 할 수 있습니다.  
예: "오늘 우산 필요해?", "주말에 등산 가도 돼?", "빨래 언제 널면 좋아?"  
자세한 예시는 [AI 어시스턴트 맑음이](AI-어시스턴트-맑음이) 페이지를 참고하세요.

### 하루 질문 횟수가 찼으면 어떻게 하나요?

무료는 하루 10회, 프리미엄은 무제한입니다.  
프리미엄으로 업그레이드하면 횟수가 늘어납니다.

### 음성으로 질문하려면 어떻게 하나요?

채팅 화면에서 마이크 아이콘을 탭하세요. 첫 사용 시 마이크 권한이 필요합니다.

## 위젯

### 위젯은 어떻게 설치하나요?

[위젯 설정 가이드](위젯-가이드) 페이지에 Android와 iOS별 상세 방법이 있습니다.

### 위젯이 날씨를 업데이트하지 않아요

- Android: 배터리 최적화에서 맑음 앱을 제외해주세요
- iOS: 앱을 한 번 열어서 수동 갱신하세요
- 두 플랫폼 모두: 앱 → 설정 → 위젯 갱신 간격 확인

## 알림

### 알림을 받지 못하고 있어요

1. 기기 설정 → 앱 → 맑음 → 알림 → 모두 허용 확인
2. 맑음 앱 → 설정 → 알림에서 원하는 항목이 켜져 있는지 확인
3. Android: 배터리 최적화에서 맑음 앱을 제외해주세요

### 알림을 끄고 싶어요

맑음 앱 → 설정 탭 → 알림 → 원하는 알림 항목의 토글을 끄세요.  
항목별로 개별 설정이 가능합니다.

## 계정 및 구독

### 여러 기기에서 프리미엄을 사용할 수 있나요?

같은 구글 플레이 또는 앱스토어 계정으로 로그인한 모든 기기에서 사용 가능합니다.

### 기기를 바꿨을 때 프리미엄은 어떻게 복원하나요?

설정 탭 → 프리미엄 → **구독 복원** 버튼을 탭하세요.

### 구독을 어떻게 취소하나요?

- Android: Google Play → 구독 → 맑음 → 취소
- iOS: 설정 → Apple 계정 → 구독 → 맑음 → 취소

취소 후에도 현재 기간 끝까지 프리미엄이 유지됩니다.

## 기타

### 여러 지역을 추가할 수 있나요?

홈 화면 상단 지역명 탭 → + 버튼으로 도시를 추가하세요. 스와이프로 지역 전환이 가능합니다.

### 앱이 배터리를 많이 소모하나요?

날씨 데이터는 앱 실행 시에만 갱신하며, 백그라운드 위치 추적을 하지 않습니다.  
위젯 갱신(30분 간격)이 소량의 배터리를 사용합니다.
