# 위젯 설정 가이드

맑음 날씨 위젯을 홈 화면에 추가하는 방법을 안내합니다.

---

## 맑음 위젯이란 무엇인가요?

홈 화면에서 앱을 열지 않고도 현재 날씨와 행동 추천을 바로 확인할 수 있는 위젯입니다.

**위젯에서 표시되는 정보:**
- 현재 기온 · 체감온도
- 날씨 상태 아이콘
- 지역명
- 행동 추천 한 줄 메시지 ("우산 챙기세요", "가디건 추천" 등)
- 선택한 아트 테마 배경

30분마다 자동으로 날씨를 갱신합니다.

---

## Android 위젯 설치 방법

### 위젯 추가하기

1. 홈 화면의 빈 곳을 **길게 누릅니다**
2. 하단 메뉴에서 **위젯**을 탭합니다
3. 앱 목록에서 **맑음(Malgeum)** 을 찾습니다
4. 원하는 크기를 선택합니다:
   - **2×2** — 소형 (기온 + 날씨 아이콘 + 한 줄 추천)
   - **4×2** — 중형 (상세 정보 + 행동 추천 카드)
5. 위치를 조정한 후 손을 뗍니다

### 위젯 크기 변경

설치된 위젯을 길게 누르면 크기 조절 핸들이 나타납니다. 드래그해서 조절하세요.

### 위젯이 업데이트되지 않을 때

- 배터리 최적화 설정에서 맑음 앱을 **제외** 처리하세요
- 설정 → 앱 → 맑음 → 배터리 → 제한 없음

---

## iOS 위젯 설치 방법

### 위젯 추가하기

1. 홈 화면을 **길게 눌러** 편집 모드로 진입합니다
2. 좌상단 **+** 버튼을 탭합니다
3. 검색창에 **맑음** 또는 **Malgeum** 을 입력합니다
4. 위젯 스타일을 선택합니다:
   - **소형** — 기온 + 날씨 아이콘
   - **중형** — 기온 + 행동 추천 + 체감온도
5. **위젯 추가**를 탭하고 위치를 조정합니다
6. 오른쪽 상단 **완료**를 탭합니다

### 잠금 화면 위젯 (iOS 16 이상)

1. 잠금 화면을 길게 누릅니다
2. **사용자화**를 탭합니다
3. 위젯 영역을 탭한 후 맑음 소형 위젯을 추가합니다

---

## 위젯 표시 지역 변경 방법

위젯은 맑음 앱에서 마지막으로 확인한 지역의 날씨를 표시합니다.  
특정 지역으로 고정하려면:

1. 맑음 앱을 열고 원하는 지역으로 전환합니다
2. 위젯이 자동으로 해당 지역으로 업데이트됩니다

---

## 위젯 아트 테마 변경 방법

앱 내 아트 테마를 변경하면 위젯 배경도 함께 바뀝니다.  
설정 탭 → 아트 테마에서 변경하세요.

---

## 자주 발생하는 위젯 문제

**위젯에 "위치 없음" 또는 날씨가 표시되지 않을 때**  
→ 맑음 앱을 한 번 열어서 날씨를 새로고침하세요. 이후 위젯이 정상 표시됩니다.

**Android에서 위젯이 자주 멈출 때**  
→ 배터리 절약 모드 또는 도즈(Doze) 모드 영향입니다. 배터리 최적화에서 맑음을 제외하세요.

**iOS에서 위젯이 오래된 날씨를 보여줄 때**  
→ 위젯은 iOS 정책에 따라 최대 30분 간격으로 갱신됩니다. 앱을 열어 수동 갱신하면 즉시 업데이트됩니다.
