## 행동 추천 카드란 무엇인가요?

행동 추천 카드는 맑음의 핵심 기능입니다.  
날씨 데이터를 분석해 "지금 무엇을 해야 하는지"를 직접 알려줍니다.

### 제공되는 추천 카드 종류

| 카드 | 조건 | 예시 메시지 |
|------|------|------------|
| 우산 | 강수확률 30% 이상 | "우산 챙기세요 (확률 60%)" |
| 옷차림 | 체감온도 기준 | "가디건 필요한 날씨예요" |
| 마스크 | 미세먼지 나쁨 이상 | "마스크 착용 권장 (나쁨)" |
| 선크림 | 자외선 지수 6 이상 | "자외선 강해요 — 선크림 필수" |
| 꽃가루 | 꽃가루 농도 높음 | "꽃가루 주의, 알레르기 조심" |
| 실내 운동 | 미세먼지 나쁨 + 비 예보 | "오늘은 실내 운동을 권장해요" |
| 세차 | 맑음 + 비 없음 3일 | "세차하기 좋은 날이에요" |
| 빨래 | 맑음 + 바람 + 습도 낮음 | "빨래 잘 마를 날씨예요" |

### 등급 시스템

- **초록** — 안전, 챙기지 않아도 됩니다
- **노란** — 권장, 챙기면 좋습니다
- **빨간** — 필수, 꼭 챙기세요
- **진빨간** — 심각, 외출을 자제하세요

## 출퇴근 날씨 비교는 어떻게 작동하나요?

출근 시간(기본 08:30)과 퇴근 시간(기본 18:00)의 날씨를 나란히 보여줍니다.

**비교 항목:**
- 기온 · 체감온도
- 강수확률
- 날씨 상태 (맑음/흐림/비)
- 한 줄 옷차림 추천

**설정 변경 방법:**  
설정 탭 → 출퇴근 시간 → 원하는 시간으로 변경

## AI 어시스턴트 맑음이 사용법

홈 화면 하단의 채팅 버튼을 탭하면 맑음이와 대화할 수 있습니다.

**이런 질문을 해보세요:**
- "오늘 우산 필요해?"
- "이번 주 가장 맑은 날은 언제야?"
- "주말에 등산 가도 돼?"
- "미세먼지 언제 좋아져?"
- "내일 아침 출근 날씨 어때?"

음성 입력 버튼을 탭해 말로 물어볼 수도 있습니다.

> 무료 버전: 하루 10회 | 프리미엄: 하루 50회

자세한 내용은 [AI 어시스턴트 맑음이](AI-어시스턴트-맑음이) 페이지를 참고하세요.

## 주간 예보는 어떻게 보나요?

홈 화면에서 아래로 스크롤하거나, 하단 탭의 주간 아이콘을 탭하세요.

**제공 정보:**
- 5일간 최고·최저 기온
- 날씨 상태 아이콘
- 일별 강수확률
- 미세먼지 등급
- 자외선 지수

---

## 아트 테마는 무엇인가요?

15가지 예술 스타일로 날씨 화면을 꾸밀 수 있습니다.  
같은 날씨도 내가 선택한 아트 렌즈로 다르게 보입니다.

**무료 테마:** 반 고흐, 모네, 클림트, 고갱, 팝아트, 바우하우스, 우키요에  
**프리미엄 테마:** 무하, 신스웨이브, 바스키아, 호크니, 리소그래프, 더블노출, 카우스, 루이스웨인

설정 탭 → 아트 테마에서 변경하세요.

## 날씨 알림 설정 방법

설정 탭 → 알림에서 6가지 알림을 개별로 켜고 끌 수 있습니다.

| 알림 | 기본 시간 |
|------|----------|
| 출근 전 옷차림 + 강수 | 출근 30분 전 |
| 오전 날씨 요약 | 매일 08:00 |
| 미세먼지 경보 | 나쁨 이상 시 즉시 |
| 자외선 경보 | 강함 이상 시 즉시 |
| 저녁 내일 예보 | 매일 21:00 |
| 강수 직전 알림 | 비 1시간 전 |

## 여러 지역 추가 방법

홈 화면 상단 지역명 탭 → + 버튼 → 도시 검색 또는 현재위치 추가  
지역 간 전환은 좌우로 스와이프하세요.
